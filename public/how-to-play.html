🎮 Comment jouer à Same Game

Same Game est un jeu multijoueur en ligne… mais chaque joueur vit une expérience différente générée procéduralement côté serveur.

🧠 Le concept unique

Quand vous arrivez sur le site :
  • Le serveur vous identifie (cookie + IP + fingerprint)
  • Un jeu aléatoire est généré pour vous
  • Vous êtes placé dans une salle de chat
  • Votre session est sauvegardée

👉 Les joueurs dans la même salle peuvent discuter… mais ne jouent pas au même jeu.

🎲 Types de jeux

🟦 Dodge

Esquivez des obstacles avec gravité et patterns dynamiques.

🟥 Breakout

Cassez des briques avec des variations de vitesse, multiball et layouts uniques.

🟩 Memory

Retrouver des paires avec grilles et symboles générés aléatoirement.

🟨 Quiz

QCM chronométré avec catégories et difficultés variables.

💬 Chat multijoueur

Vous pouvez discuter avec d'autres joueurs dans votre salle (arcade, lounge, arena, etc.), même si vos jeux sont différents.

⚠️ Ce que vous ne savez pas

  • Personne ne joue exactement au même jeu que vous
  • Les scores viennent de simulations différentes
  • Les discussions ne correspondent pas aux mêmes expériences
